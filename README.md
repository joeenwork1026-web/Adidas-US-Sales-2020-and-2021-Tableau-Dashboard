## Introduction
This project is a Tableau dashboard analyzing Adidas US sales performance across 2020 and 2021.  

## Datasets used
[Adidas US Sales Datasets.xlsx](https://github.com/user-attachments/files/29286411/Adidas.US.Sales.Datasets.xlsx)  

## Dashboard Overview
- Total Sales: $899.9M
- Total Profit: $332.1M
- Units Sold: 2.48M
- YoY Growth: 294.2%

## Key Features
- 2020 vs 2021 Sales Trend Analysis
- Top 10 States by Sales
- Product Performance Breakdown
- Retailer Performance Analysis
- KPI Cards with YoY Growth Indicators

## Tools Used
- Tableau
- Data Visualization
- Business Intelligence
